Machine Learning, um processo que cria uma inteligencia artificial a partir de uma gama de dados machinelearning. Esse processo pode ser feito de várias formas, porém a que será abordada é o método Deep Learning utilizando uma rede neural convulacional CNN. Um método que tem como objetivo criar aplicações que possam aprender e realizar tarefas de forma autônoma.

O nome "deep learning" vem da estrutura utilizada para realização do método que consiste em várias camadas de neurônios artificiais conectados, permitindo que o sistema aprenda representações mais complexas dos dados de entrada, podendo ser desde um conjunto de inteiros para fazer uma predição de calculo até um sistema de navegação de um veiculo Tesla  deeplearning.

Neurônios artificiais são a principal composição da rede neural. Eles são estruturas agrupadas em camadas, inspiradas nos neurônios biológicos presentes no cérebro humano. Cada neurônio artificial recebe várias entradas (representadas por valores numéricos), que são multiplicadas por um peso determinado e somados. Esse resultado então é processado através de um calculo que chamamos de função de ativação, que determina a saída, que é passada para os outros neurônios de uma rede.

A CNN é uma estrutura composta por diversos neurônios e com camadas de convolução(camadas definidas com um conjunto de filtros que, por sua vez, são matrizes que definem uma determinada característica visual para detectar na imagem). Sua arquitetura pode ser dividida em três camadas principais:  Input Layer (camada de entrada), Hidden Layers (camadas escondidas) e Output Layers (camada de saída), com as funções de entrada de dados, processamento e saída respectivamente cnn, medical.

A primeira camada, Input Layer, é a responsável por receber os dados base (fotos, vídeos) e transforma-los em dados matemáticos e envia-los para as camadas seguntes cnn. Já as Hidden Layers são camadas que possuem funções matemáticas não lineares, em que, cada uma e modelada para retornar um valor específico de acordo com os dados de entrada para então envia-los para a ultima parte da rede. Por fim as Output layers produz o resultado final de acordo com as predições (resultados) das camadas anteriores.
